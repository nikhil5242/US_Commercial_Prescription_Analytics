# Dataset Download Instructions

The datasets used in this project are publicly available from the Centers for Medicare & Medicaid Services (CMS).

The original datasets are not included in this repository because they exceed GitHub's file size limits.

Datasets used in this project:

1. Medicare Part D Prescribers by Provider
2. Medicare Part D Prescribers by Geography and Drug
3. Medicare Part D Spending by Drug

Official Source:

https://data.cms.gov

Download the latest available datasets from the CMS Data Portal before running this project.

The SQL scripts and Python notebook are designed to work with these datasets after import into PostgreSQL.
